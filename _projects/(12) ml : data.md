<h3><code><a name="1"> 1. Reinforcement Learning for Resource Management</a></code></h3>
<br/>

<h4><code>Description</code></h4>
<p>
<code>
This project has been motivated <b>to improve emulation resource management with reinforcement learning and find the optimal solution</b>. The heuristic based approach has improved resource utilization much, however, the emulation environment cases are too diverse to define algorithms and polcies depending on different situations. <b>Double Deep Q Network (DDQN)</b>, which is one of the basic algorithm by Deep Mind, has been applied to solve this problem.</code>
</p>
<p><code>
To acheive the application for DDQN, the simulation environmnet has been developed, and simulation data has been generated from the previous job data. From the experiment, it has been shown that algorithm based on reinforcement learning has been better performance than no policy.
</code></p>
